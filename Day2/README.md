## 🧠 Key Learning

| Category | Details |
|---|---|
| Core Idea | A vague prompt → a vague answer. A specific, structured prompt → a focused, usable answer |
| Weak Prompt | "Tell me about marketing." |
| Engineered Prompt | "Act as a marketing expert. Explain 3 beginner social media strategies for a small bakery, with an example each." |
| Why It Works | Adding role + audience + count + format turns a generic response into a structured, professional one |
| Benefits | Saves Time, More Accurate, Professional Output |
